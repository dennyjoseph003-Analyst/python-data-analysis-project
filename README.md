# 🚀 Python Data Analysis Project  

📊 This project demonstrates data cleaning, preprocessing, and analysis using **Python, pandas, and NumPy**.  
🔍 It focuses on merging datasets, handling missing values, and generating insights through reporting dashboards.  

## ✨ Key Features  
- 🧹 Data Cleaning & Validation  
- 📈 Exploratory Data Analysis  
- 📊 KPI Tracking & Reporting  
- 🎨 Visualization with Power BI & Python  

## 🛠 Skills Highlighted  
- 🐍 Python (pandas, NumPy)  
- 📑 Excel (Pivot Tables, VBA)
